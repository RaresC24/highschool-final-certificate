# High School Final Certification Project: Movie Hub

This repository contains my final project for the 12th-grade computer science certification (Atestat). The primary objective of this project was to demonstrate foundational proficiency in core web technologies: HTML, CSS, and basic JavaScript.

## Project Overview

The project is a static front-end website dedicated to movies, featuring categorized film lists, top rankings, and profile layouts. 

At the time of development, database architectural concepts and dynamic data fetching had not yet been integrated into my skillset. Consequently, the website relies entirely on static pages. Every movie entry, description, image link, and category list was manually inputted and hardcoded directly into the HTML structure.

*Note: Because this project was created quite some time ago, some external image URLs or links used for the movies may no longer be active and might fail to load.*

## Repository Structure

The website is divided into multiple thematic pages to organize the content, styled via independent CSS files, and enhanced with simple JavaScript functionalities.

### Core Pages
- `index.html`: The main landing page introducing the website.
- `tipuri_filme.html`: A navigational hub displaying the different available movie genres.
- `top_10.html`: A dedicated page showcasing a customized Top 10 movies list.
- `profil.html`: A layout demonstrating a user profile interface.
- `contact.html`: A page featuring a contact form template.

### Genre-Specific Pages
To demonstrate structural layouts and handle volume, the project contains several pages corresponding to specific movie genres. Each of these lists manually curated movies:
- `filme_actiune.html` (Action)
- `filme_comedie.html` (Comedy)
- `filme_drama.html` (Drama)
- `filme_horror.html` (Horror)
- `filme_mister.html` (Mystery)
- `filme_romance.html` (Romance)
- `filme_SF.html` (Sci-Fi)

### Assets & Styling
- **CSS:** Separated styling files for different domains (e.g., `style.css`, `filme_layout.css`, `profil.css`, `contact.css`) to maintain organized styling rules.
- **JavaScript:** Features basic client-side scripts (`script.js`, `filme_layout.js`, `profil.js`, `tipuri_filme.js`) used to handle simple user interactions and DOM manipulation.
- **Documentation:** The repository includes the original written documentation concepts (`documentatie_atestat.pdf` and `.docx`) that were required for the certification defense.

## Technical Scope

This project serves as an archive of early-stage front-end web development, utilizing:
- **HTML5:** Semantic structuring, grid layouts, tables, and forms.
- **CSS3:** Manual layouting, styling, and custom animations designed to create a visually pleasing user experience.
- **Vanilla JavaScript:** Simple interactivity features.
